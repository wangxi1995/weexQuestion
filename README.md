### weex简介
```
1.h5页面，安卓，ios共用一套代码
2.h5的写法创建页面，安卓和ios调取页面的链接
3.h5调用原生的组件，安卓和ios负责拓展功能

weex的优势
1.调用的原生组件所以用户体验度好
2.h5开发速度快，节约成本

weex的劣势
1.文档不健全，学习难度大
2.坑点较多
```
